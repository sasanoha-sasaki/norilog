目的
====
webブラウザーでコメントを投稿するwebアプリケーションの練習

ツールバージョン
================
:python: 3.10
:pip:    

インストールと起動方法
===================
$git clone https://github.com/beproud/norilog
cd norilog
python3.6 -m venv venv
source venv/bin/activate
(venv)$ pip install
(venv)$ norilog
*Running on http://127.0.0.1:8000/
